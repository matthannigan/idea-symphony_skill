## Background

### Context

The Tier 1 roster (medium effort, 8 personas) is established with:
- **Human-centered cluster:** Storyteller (narrative/experiential) and Appreciative Inquirer (strengths-based/aspirational)
- **Question:** Can a 9th persona add value without redundancy?

Both Audience Advocate and Empath are classified as human-centered personas. Prior research (I04) validated both as distinct from each other. The question here is not "are they different?" but rather "do they both add marginal value to an already-diverse Tier 1 set?"

### Key Differences

**Audience Advocate:** Focuses on structural barriers, equity, accessibility, stakeholder needs, and "How Might We" framing. Primary mode is uncovering user pain points and designing for inclusivity.

**Empath:** Focuses on emotional resonance, gut feelings, subjective experience, and affective dimensions. Primary mode is emotional excavation including both positive and negative emotions.

**Existing Tier 1 human-centered coverage:**
- Storyteller: narrative structure, lived experience, journey arcs
- Appreciative Inquirer: strengths-based framing, positive aspirations, building on existing success

### Research Questions

1. **RQ9a Unique territory:** What questions does each candidate generate that NO Tier 1 persona covers?
2. **RQ9b Storyteller overlap:** Given Storyteller is already in Tier 1, how much does each candidate overlap with it?
3. **RQ9c Appreciative Inquirer overlap:** Given AI is already in Tier 1, how much does each candidate overlap with it?
4. **RQ9d Dimension coverage:** Which dimensions does each candidate add to the Tier 1 aggregate? (Strategic, Tactical, Creative, Analytical, Human-centered, Systems, Provocative)
5. **RQ9e Selflessness:** Do the candidate's questions open territory for diverse Phase 3 personas or primarily for human-centered answerers?
6. **RQ9f Topic sensitivity:** Does either candidate show strong topic-dependent performance (e.g., AA much better on community topics, Empath better on personal topics)?
7. **RQ9g Marginal value:** Net new question territory added vs total question volume increase

**Comparison criteria:**

1. **Unique territory:** What questions does each candidate generate that NO Tier 1 persona covers? Look for themes, framing styles, and question modes not present in the 8-persona aggregate.

2. **Overlap with Storyteller:** Since Storyteller is already in Tier 1, count questions from each candidate that overlap with Storyteller's narrative/experiential framing. Calculate convergence %.

3. **Overlap with Appreciative Inquirer:** Since AI is already in Tier 1, count questions from each candidate that overlap with AI's strengths-based/positive aspirational framing. Calculate convergence %.

4. **Dimension coverage gap:** Classify each candidate's questions by dimension (Strategic, Tactical, Creative, Analytical, Human-centered, Systems, Provocative). Which dimensions does each add or strengthen in the Tier 1 aggregate?

5. **Selflessness:** For each candidate's questions, identify which Phase 3 personas could productively answer it. High-value Phase 2B questions open territory for diverse personas, not just human-centered ones. Calculate % selfless questions.

6. **Topic sensitivity:** Does either candidate show strong topic-dependent performance? Extract performance scores per topic and identify patterns (e.g., AA stronger on community/tool-library topics vs Empath stronger on personal/habit-tracker).

7. **Marginal value:** Calculate net new question territory added (% of questions that don't overlap with Tier 1 + Storyteller + AI) vs total volume increase.
